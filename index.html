Font-size у родительского класса 20px

Селекторы

Существуют селекторы по классу:

Пример использования
HTML CSS Комментарий
<p class="warning">
</p> 		.warning { background:red }
<p class="warning border-gold">
</p> 		.border-gold { border:1px solid yellow } Можно задавать несколько классов и использовать один класс из нескольких
<p class="warning quot;> <li>test</li>
</p> 		.warning li { color: green } Можно применять стили к атрибутам внутри класса

Так же, существуют селекторы по ID, обозначаются в css: #idselektor

Можно указать в css .class1 .class2, и тогда правило применится к элементам с классом "class2", предком которого является элемент с классом "class1"

Если необходимо, можно выбрать элемент, у которого присутствуют сразу несколько классов, указав их без пробела. Например: .class1.white

Дочерний селектор .class1 > li используется только для детей-элементов, не используясь в дальнейших потомках.

Существует большое количество селекторов с родительскими связями.

Существуют селекторы по атрибуту:

HTML CSS

</div> 		p[atribut="res"] {
background: gray;}
<div>

Ссылки

При добавлении ссылок, можно указать атрибут "a href = "..." target со значениями:
_blank Загружает страницу в новое окно браузера.
_self Загружает страницу в текущее окно.
_parent Загружает страницу во фрейм-родитель, если фреймов нет, то это значение работает как _self.
_top Отменяет все фреймы и загружает страницу в полном окне браузера, если фреймов нет, то это значение работает как _self.

Блочные элементы

<p>, <div>, <h2>, <ul>, и др.

Они занимают все доступное место по горизонтали.
Каждый новый блочный элемент начинается с новой строки.
Блочным элементам можно задавать любые размеры: высота, ширина, внутренние и внешние отступы.

Если блок меньше содержимого, есть свойство overflow:(hidden|visible)

Строчные элементы

Строчным элементам нельзя задать ширину или высоту. Их размер зависит от содержимого.
Так же нельзя задвать вертикальные внешние отступы

Строчным элементам можно задавать внутренние отступы, а так же внешние горизонтальные

Если разместить несколько строчных элементов, то они будут с расстоянием между друг другом. Это происходит потому, что браузер распознает элементы "как отдельные слова".

Текст 1 Текст 2 Текст 3

Чтобы этого избежать, нужно задать размер шрифта отдельным строчным элементам, затем все отдельные элементы поместить внутрь одного блокого, размер шрифта которого будет равен 0

Текст 1 Текст 2 Текст 3

HTML CSS
<div class="stroka">
  <span>Текст 1</span>
  <span>Текст 2</span>
  <span>Текст 3</span>
</div> 		span { border: 1px solid black; font-size: 14px; } .stroka { font-size: 0 }

Можно принудительно сделать строчный элемент блочным, добавив в CSS display:block

Можно создавать блочно-строчные элементы.
display:inline-block
Теперь можно задавать размеры и отступы.

Блочно-строчные элементы отличаются от других тем, что они похожи на блочные, но распологаются в одну строку

Float

CSS свойство float указывает, что текущий элемент должен быть изъят из обычного flow (потока) и прижат к левой или правой стороне родительского элемента. Текст и inline элементы будут обтекать такой элемент.

Для того чтобы избежать вываливания из родительского элемента, нужно указать на родительском элементе класс clearfix, с кодом:
.clearfix::after {
clear:both;
display:table;
content: "";
}

FlexBox

Это CSS механизм, который позволяет контролировать размер, порядок и выравнивание по нескольким осям.(слева-направо, сверху-вниз)

Чтобы создать flexbox, необходимо задать родительскому контейнеру свойство display:flex

HTML CSS
<section>
<div class="flex">1</div>
<div class="flex">2</div>
<div class="flex">3</div>
<div class="flex">4</div>
<div class="flex">5</div>
</section> 		.flex {
width: 40px;
height: 40px;
background: lime;
border: 1px solid;
}
section {
display: flex;
}

align-content всегда важнее, чем align-item, поэтому можно использовать align-content (спецификация новых версий браузера, в старых версиях может конфликтовать)

Если Flex-элементу задать margin(отступ):auto то все элементы после данного будут выравнены справа. Даже при использовании justify-content: center

Флекс-элементы могут растягиваться с помощью flex-grow: 1[до тех пор, пока не закончится свободное пространство]|2,3...(и другие доли)
За сжатие отвечает параметр flex-shrink:0[запрет на сжатие]|2,3...(и другие доли)

Можно задать базовый размер (в зависимости от основной оси) flex-basis:100px

flex-grow: 1;
flex-shrink: 0;
flex-basis: 100px;

Можно сократить до:
flex: 1 0 100px;
